# Quiz Template

This document provides the standard template for chapter quizzes in the Physical AI & Humanoid Robotics textbook.

## Quiz Structure

Each chapter quiz should contain the following question types with the specified minimum quantities:

- Multiple Choice Questions: 5-7 questions
- Practical Application Questions: 2-3 questions
- Code Analysis Questions: 2-3 questions
- Conceptual Understanding Questions: 3-5 questions

## Question Format

### Multiple Choice Questions
```
**Question [X]**: [Question text]
A) [Option A]
B) [Option B]
C) [Option C]
D) [Option D]

**Correct Answer**: [Letter]
**Explanation**: [Detailed explanation of why this is correct]
```

### Practical Application Questions
```
**Question [X]**: [Scenario-based question requiring application of concepts]

**Expected Answer**: [What the student should understand/demonstrate]
**Scoring Criteria**: [How the answer will be evaluated]
```

### Code Analysis Questions
```
**Question [X]**: [Provide code snippet and ask for analysis]

**Expected Answer**: [What the student should identify about the code]
**Key Points**: [Important elements to look for]
```

### Conceptual Understanding Questions
```
**Question [X]**: [Concept-focused question requiring explanation]

**Expected Answer**: [Comprehensive explanation expected]
**Evaluation Points**: [Criteria for assessing understanding]
```

## Scoring and Evaluation

- Each quiz should have a minimum of 10 questions
- Passing score is 80% (8 out of 10 correct for basic quizzes)
- Partial credit may be given for conceptual questions if reasoning is sound
- Answer keys must be provided with detailed explanations

## Quality Standards

- Questions must directly relate to chapter content
- Distractors in multiple choice should be plausible but clearly incorrect
- Questions should test understanding, not just memorization
- Practical questions should reflect real-world scenarios
- Code questions should use examples similar to those in the chapter
